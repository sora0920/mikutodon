# mikutodon
mikutterでmastodonできるプラグイン
# これなに
mikutterでMastodonやれるやつ
# インストール方法
1. これを叩く
```
mkdir -p ~/.mikutter/plugin
cd ~/.mikutter/plugin
git clone git@github.com:sora0920/mikutodon.git
```
2. お好きな方法でGemを突っ込む
```
gem "nokogiri"
```
3. Mastodon Worldが追加されているのでWorldを追加からMastodonを選択して追加する。
4. 抽出タブのデータソースが増えてるよ！やったねミクちゃん！
